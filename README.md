# Tapestry

A multi-model database — **graph**, **columnar/analytics**, **vector**, and
**timeseries** — with built-in **ETL** (discover, ingest, transform source data) and
**native intelligence** (natural-language → query, LLM-assisted ETL, intent-based
rules).

- **Humans:** the documentation site lives in [`web/`](web/).
- **Agents & contributors:** start at [`AGENTS.md`](AGENTS.md). The knowledge base is
  [`docs/`](docs/); how it's organized is
  [RFD-0001](docs/decide/rfd/0001-documentation-system.md).

> Early and under active construction.
