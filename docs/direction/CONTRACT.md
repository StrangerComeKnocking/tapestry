# `direction/` — area contract

**Job:** durable product intent.

**Belongs:** vision, strategy, positioning, principles, strategic bets
(`kind: direction` + `tag`), and the `roadmap` (a **pointer**: `source: summary`,
`mirrors:` the live planning source).

**Does NOT belong:**
- dated delivery plans → link the live planning source, don't copy it
- a dated bet's *rationale* → `decide/`
- metric definitions → `use/reference/metrics/`
